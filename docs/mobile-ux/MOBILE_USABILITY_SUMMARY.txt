=============================================================================
  PASSWORD VISIBILITY TOGGLE ISSUES
=============================================================================

CURRENT IMPLEMENTATION:
  <Button
    type="button"
    variant="ghost"
    size="sm"
    className="absolute right-0 top-0 h-full px-3 py-2 hover:bg-transparent"
    ...
  >

PROBLEMS:
  1. Absolutely positioned - hard to tap on mobile
  2. Overlaps input field - reduces touch target
  3. Could be accidentally triggered while typing
  4. Size is h-full but button is only sm variant (h-8)

RECOMMENDED FIX:
  <div className="relative">
    <Input
      id="password"
      type={showPassword ? "text" : "password"}
      className="pr-10"  // Add padding for toggle
      ...
    />
    <Button
      type="button"
      variant="ghost"
      size="icon-sm"
      className="absolute right-1 top-1/2 -translate-y-1/2"  // Better centering
      ...
    >
